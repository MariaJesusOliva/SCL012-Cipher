MeToo

¿Qué es MeToo?

Es una app para que mujeres y disidencias puedan compartir un mensaje en RRSS, principalmente grupos públicos de los lugares donde transitará o está para ver si alguna se encuentra cerca o hará el mismo tramo para ir juntas y mas seguras.

El código se define por el día que lo escribes. Ej: para 25/02 es 25. para 03/12 es 03.
